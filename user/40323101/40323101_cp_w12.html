第十二週作業

fix.bat

1.下載copy.com的fix的zip


2.打開Ana3_2015資料夾中的Scite


3. 把start.bat和剛剛解壓縮完的fix.bat在Scite打開


4. 到google搜尋leo-editor github,下載最新更新的fix


5. 把剛剛載下來的檔案丟到一開始下載的fix資料夾中


6.打開Scite,將fix.bat新增一行copy leo-editor-broke-abbrev.zip C:\Ana3_2015\data\


7.再點start.bat,找到第35行改成pip install leo-editor-broke-abbrev.zip


8.2個都要按儲存,完畢之後啟動fix再點start,這樣就把leo變成最新版本了


心得與自評


這禮拜是把leo editor更新成最新的5.1版本,老師第一次講的時候,聽得不是很懂, 不過還好同學都有錄影片再講解,看完影片,自己再跟著步驟跟著試一遍, 反正每次上課前都要做這件事,所以應該可以越練越順手。
