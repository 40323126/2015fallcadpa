2015 電腦輔助設計實習 2

週六 19 九月 2015

在github中，建立自己的github pages 並且可以顯示在(user.github.io/倉儲) 做法: git clone 下載遠端的資料庫 clone完 在用cd 進入資料庫 利用git branch gh-pages建立分支 git checkout gh-pages切換分支 將老師的reveal.js 整包複製到分支已經切換到的倉儲中 在後再上傳 上傳指令 git add -A git commit -m"40323116" git push origin gh-pages
